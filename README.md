# トルコリラ円スワップ運用シミュレーター

TRY/JPYを長期保有した場合のスワップ損益、為替下落による評価損益、合計損益を概算する静的Webアプリです。

## 主な機能

- TRY/JPYの現在価格を入力
- 1万通貨あたりの1日スワップポイントを入力
- 年間の為替下落率を入力
- 年間のスワップ低下率を入力
- スプレッド費用を入力
- 入金額とレバレッジから初期保有数量を計算
- 日次複利モードのオン・オフ
- 評価損益、スワップ損益、合計損益をグラフ表示
- 年ごとの結果を表で表示

## 複利モードの考え方

複利モードをオンにすると、毎日のスワップ収益、評価損益、スプレッド費用を反映した実質証拠金に対して、設定レバレッジまでTRY/JPYを買い増します。

スプレッド費用は、初回購入と買い増しのたびに発生する費用として計算します。初期設定では、1.4pips、0.1pipsあたり1万通貨で100円の費用としています。

為替下落率とスワップ低下率は、1年かけて一次関数的に減少する前提です。

税金、ロスカット、証拠金維持率、FX会社ごとの細かいルールは含めていません。

## 使い方

`index.html` をブラウザで開くと使えます。

Vercelに公開する場合も、まずはこのフォルダをGitHubにアップロードして、そのリポジトリをVercelでImportすれば公開できます。
